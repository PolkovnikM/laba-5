# Лабораторная работа №5 по Agile и Scrum
**Проект:** Генератор и валидатор паролей

**Описание:** 
Веб-приложение на Flask для генерации случайных паролей и проверки их надежности. 
Пользователь может указать длину пароля, сгенерировать его и проверить существующий пароль на безопасность. 
Проект демонстрирует базовые принципы веб-разработки и безопасность паролей.

**Технологии:** Python, Flask, HTML
